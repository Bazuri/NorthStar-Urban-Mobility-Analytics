# NorthStar Urban Mobility and Logistics
## Databases and Analytics Assignment — University of West London

### Project Overview
This repository contains the analytical notebooks for the NorthStar 
Urban Mobility case study. The analysis covers SQL in R, Python data 
processing, MongoDB database design, and query optimisation.

### Repository Structure

| Notebook | Description |
|---|---|
| NorthStar_Step3_SQL_in_R.ipynb | SQL queries inside R using sqldf |
| NorthStar_Step5_Python.ipynb | Python data processing using Pandas and NumPy |
| NorthStar_Step6_MongoDB.ipynb | MongoDB Atlas database design, CRUD operations, and query optimisation |

### Technologies Used
- R (sqldf, ggplot2, dplyr, lubridate)
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- MongoDB Atlas (PyMongo)
- Google Colab

### Dataset
NorthStar dataset containing 9 CSV files covering customers, 
orders, deliveries, drivers, vehicles, hubs, complaints, 
incidents, and app events.
